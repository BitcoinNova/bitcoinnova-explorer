How to Bootstrap the Bitcoin Nova Blockchain

This guide will help you install a recent copy of the blockchain. This should significantly speed up the task of getting the blockchain synced up so you can use your wallet.

Windows:

  1. Make sure Bitcoinnovad.exe, Bitcoinnova-service.exe, and/or the GUI wallet are not running.

  2. Open File Explorer and type %APPDATA%\bitcoinnova and hit enter.

Note: if the folder doesn't exist, just go to %APPDATA% instead and create a folder named bitcoinnova.

  1. Delete the following if they exist:

  • blockindexes.bin

  • blocks.bin

  • "DB" folder

    Note: In case it is unable to delete the files due to them being used by some other program, follow these steps:

    • Open Task Manager with the shortcut Ctrl + Shift+ Escape.
    • Click on Processes.
    • Click on Image Name.
    • Scroll to the bottom.
    • Click on Bitcoinnova-service.exe.
    • Click on End Process.
    • Click on End Process again.
    • Try to delete them again.

Mac & Linux:

  1. Make sure Bitcoinnovad, Bitcoinnova-service, and/or the GUI wallet are not running.

  2. Open Finder.

  3. Use the shortcut Command + Shift + G to bring up Go to Folder:

  1. Delete the following if they exist:

  • blockindexes.bin

  • blocks.bin

  • "DB" folder

  1. Download the latest snapshot of the blockchain.

  2. Move the two new downloaded files, blockindexes.bin and blocks.bin into the ~/.bitcoinnova/ folder.

  3. Start Bitcoinnovad or the GUI like you normally do.

  4. See the Expected Results section below.

Expected Results if Done Correctly

When you start Bitcoinnovad you should see this. Note that the blocksize (150246) in this example will be a different number:

with index 1000 / 150246 2018-Feb-01 18:43:38.258202 INFO Imported block with index 2000 / 150246 2018-Feb-01 18:43:38.928033 INFO Imported block with index 3000 / 150246 2018-Feb-01 18:43:39.454094 INFO Imported block with index 
4000 / 150246 2018-Feb-01 18:43:40.142969 INFO Imported block with index 5000 / 150246 2018-Feb-01 18:43:40.830674 INFO Imported block with index 6000 / 150246

After it completes it will start syncing incrementally like so:

2018-Feb-01 18:47:48.075930 INFO Imported block with index 150000 / 150246 2018-Feb-01 18:47:48.860470 INFO Core initialized OK 2018-Feb-01 18:47:48.860470 INFO Initializing p2p server...

If you are using the GUI wallet, you can instead view the progress by opening Bitcoinnova-service.log and scrolling to the bottom.
